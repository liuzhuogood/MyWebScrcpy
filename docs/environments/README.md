## 脱敏与保密约定

- 本目录含敏感信息（服务器 IP、账号、连接方式），仅供项目内部使用。
- 不要把敏感信息写入 git 提交备注，不要粘贴到外部服务，不要自动发布到 ima。
- 明文密钥优先存放配置中心或环境变量，文档只标明存放位置。
- 如需整个目录不纳入版本管理，把 `docs/environments/` 加入 `.gitignore`；默认随仓库提交，但提交前请确认已脱敏。
