# v1.2.0 版本说明

## 版本信息

- **版本号**：v1.2.0
- **发布日期**：2026-07-xx
- **版本类型**：功能版本（Feature）

## 变更内容

### 新增功能

1. **脚本管理**
   - 新增 `/scripts.html` 页面，支持按「分类」组织 Python 脚本
   - 分类管理：新建分类、重命名分类、删除分类（连带删除其下脚本）
   - 脚本管理：在分类下新建、编辑、重命名、删除脚本
   - 在线代码编辑器（CodeMirror 6，Python 语法高亮 / 行号；离线降级 textarea）
   - 脚本元数据：描述、参数（为后续执行传参预留），创建/更新时间自动记录
   - 脚本以独立 `.py` 文件落盘到运行目录 `scripts/`，可 Git 管理、可直接执行

2. **设备列表页入口**
   - 在 `index.html` 头部添加「脚本管理」按钮，与「大屏模式」并列

### 技术细节

- 存储：文件式（`scripts/<分类>/<脚本>.py`），无数据库依赖，元数据用脚本头部 `# @key value` 注释块承载
- 后端：新增 `internal/scripts` 包，注册 `/api/scripts/*` REST 路由；严格的路径白名单校验防目录穿越
- 工作目录可通过环境变量 `SCRIPTS_DIR` 覆盖，便于容器/部署外挂
- 为后续「脚本执行」与「AI 生成脚本」预留元数据结构

### 兼容性

- 不影响现有 `index.html`、`player.html`、`dashboard.html` 功能
- 不修改现有 Go 后端逻辑（仅新增路由与包）
- 脚本目录为运行时用户数据，升级/回滚均不影响投屏主流程

### 非目标（留待后续版本）

- 不在服务端执行脚本（python3 运行与输出捕获）
- 不接入 AI 生成脚本
- 不做多用户/权限/鉴权
